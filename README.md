## Installation

### Prerequisites

- Bundler
- Docker (for SonarQube code coverage only)
- Homebrew (for SonarQube code coverage only)

### Installation

```
bundle install
```

### Local development/writing

```
bundle exec rake
```

### Build for Production

```
bundle exec rake build:generate
```
